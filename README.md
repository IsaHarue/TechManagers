

# TechManagers <img width="100" height="100" alt="image" src="https://github.com/user-attachments/assets/d5f791d7-66d8-48ce-b576-8ed98747d728" />


Sistema web desenvolvido para gerenciamento de uma loja, permitindo o controle de estoque de produtos, funcionários e categorias de itens em uma única plataforma.

## 📖 Sobre o projeto

O **TechManagers** é um sistema desenvolvido com **Python** e **Flask** que auxilia na administração de uma loja por meio do gerenciamento de produtos, funcionários e relatórios.

A aplicação permite cadastrar, editar, visualizar e excluir informações de forma prática, organizando os produtos em diferentes categorias para facilitar o controle do estoque.

---

## 🚀 Tecnologias utilizadas

- Python
- Flask
- SQLite
- HTML5
- CSS3

---

## ✨ Funcionalidades

### 🔐 Autenticação

- Login de usuários.

### 📦 Gestão de Produtos

- Cadastro de produtos;
- Edição de informações;
- Exclusão de registros;
- Visualização de detalhes;
- Pesquisa de produtos.

### 🗂️ Categorias

Os produtos são organizados nas seguintes categorias:

- 👕 Roupas
- 🔨 Ferramentas
- 🧵 Matéria-Prima

Cada categoria possui sua própria tela para facilitar o gerenciamento.

### 👥 Gestão de Funcionários

- Cadastro de funcionários;
- Edição de informações;
- Exclusão de registros;
- Visualização de detalhes.

### 📊 Relatórios e Dashboard

- Relatórios do sistema;
- Dashboard com gráficos para acompanhamento das informações cadastradas.

## ▶️ Como executar o projeto

### Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/TechManagers.git
```

### Acesse a pasta

```bash
cd TechManagers
```

### Instale as dependências

```bash
pip install -r requeriments.txt
```

### Execute a aplicação

```bash
python app.py
```

### Acesse no navegador

```
http://127.0.0.1:5000
```
